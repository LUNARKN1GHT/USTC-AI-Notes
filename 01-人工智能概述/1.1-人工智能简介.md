## 人工智能的定义

教材对人工智能的定义是**分层定义**的

1. 字面意思：人工创造的一个智能体。
2. 人工智能是一门研究如何使用机器复现智能的学科。
   1. 机器：当前人工智能的载体就是电子计算机。
   2. 我们通常把“机器”和用“机器”建立的人工智能系统，包括人类本身，统一抽象为“智能体”。
      1. 自治性：智能体根据对环境的感知和自身的状态与机理独立自主地执行任务或达成目标。
      2. 灵活性：知恩感题在接收到环境变化时能及时作出相应和调整，确保任务顺利执行或达成目标。
      3. 前瞻性：主动根据任务或目标进行规划和行动。
   3. 复现
      1. 复现的原理
         1. 效果驱动：只要结果能够表现出智能即可-> 弱人工智能
         2. 原理驱动：实现智能的原理来复现原理 -> 强人工智能
      2. 范围和普适性：
         1. 专用弱人工智能：只能用于某一特定的任务。
         2. 通用人工智能：即俗称 AGI
      3. 机器和人类的比重：
         1. 纯人类（L0）
         2. 人类主导（L1）
         3. 人机耦合（L2）
         4. 机器主导（L3）
         5. 纯机器（L4）
      4. 复现的效果：智能体在某任务上相对人类的能力。
         1. 落后（E0）
         2. 接近（E1）
         3. 达到（E2）
         4. 超越（E3）
         5. 最优（E4）
3. 智能的能力：即处理知识的能力。
   1. 表示：机器如何编码和存储「知识」，，即
   2. 学习：机器如何获取和更新知识，即怎么来和怎么变的。
   3. 推理：机器如何运用知识来解决问题，即怎么用。
4. ==人工智能四个要素==
   1. 算法：更准确地说是模型。
   2. 算力：更准确地说是计算，包括计算资源、如何进行计算
   3. 数据：智能体用到的有效信息
   4. 知识：主要是 $\theta$ 以及部分的 $f$，所关心的是智能体起决定性要素的内容。
5. ==人工智能学习种类==：按照数据特性分类（教材后续会对这些学习方法进行展开论述）。
   1. 有监督学习：输入、输出、反馈，这三者完整
   2. 无监督学习：没有反馈数据。
   3. [[07-强化学习|强化学习]]：反馈并不及时
   4. [[06-自监督学习|自监督学习]]：将输出直接变为输入，反馈是自身。

## 人工智能子领域

1. 按照智能行为的新颖程度来划分
   1. 基础智能：反应
   2. 分析智能
   3. 创新智能
2. 智能体处理信息的步骤
   1. 感知智能
      1. 计算机视觉
      2. 语音识别与处理
      3. 自然语言处理
      4. 多模态信息处理
   2. 认知智能
      1. 知识计算
      2. 机器学习
      3. 多智能体系统
      4. 不确定性
      5. 约束满足问题
   3. 决策智能
      1. 人工智能
