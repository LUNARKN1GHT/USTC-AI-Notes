# Q 学习

==Q学习是一种通过迭代更新动作-价值函数（Q函数）来寻找最优策略的无模型、离线策略的强化学习方法。==作为强化学习领域最经典且广泛应用的时序差分算法，它奠定了值函数逼近的理论基础并启发了深度强化学习的发展。

## 什么是 Q 学习？

### Q学习的基本概念

Q学习算法是一种经典的[[07-强化学习|强化学习]]算法，用于解决具有明确奖励的[[7.2-马尔可夫决策过程|马尔可夫决策问题]]，其核心思想是通过与环境交互自助学习最优决策策略。该算法**无需预先了解环境动态模型**，仅需基于实际交互获得的经验数据进行学习。在实现过程中，Q学习通过维护一个动作-价值函数（Q函数）来评估在特定状态下采取不同动作的长期收益，并采用时间差分方法迭代更新该函数。

### 动作-价值函数（Q函数）

Q函数，也称动作-价值函数，是一种特殊的价值函数。Q函数用来评估每个状态和行为的预期奖励（即Q值）。Q值越大，说明在该状态下采取相应行动越有利。**Q学习通过迭代地更新Q值来逼近最优Q函数**。

## Q学习算法

### 算法更新规则

贝尔曼方程描述了一个状态的价值可以通过即时奖励和未来价值的综合来计算。对于Q学习来说，贝尔曼方程用来计算最优策略下的Q值：

$$
Q^*(s, a) = R(s, a) + \gamma \sum_{s'}P(s' \mid s, a)V^*(s')
$$

更新公式如下：

$$
Q(S_t, A_T) \leftarrow Q(S_t, A_t) + \alpha \left[R_{t + 1} + \gamma \max_{a}Q(S_{t + 1}, a) - Q(S_t, A_T) \right]
$$

### 算法流程

1. 初始化 Q 表：创建一个 Q 表，通常初始化为 0
2. 选择动作：在每个时间步骤中，智能体根据当前状态和 Q 表选择一个动作这通常涉及探索和利用的平衡，以确保在学习过程中不断地探索新的动作策略。
3. 执行动作：智能体执行所选择的动作，并观察环境的相应，包括获得的奖励信号和新的状态
4. 更新Q值：根据观察到的奖励信号和新的轧辊台，智能体更新Q值，这涉及到上面提到的Q学习更新规则，即贝尔曼方程。
5. 重复迭代：智能体不断地执行上述步骤，与环境互动，学习和改进Q函数，直到达到停止条件。
