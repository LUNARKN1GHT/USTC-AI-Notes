## 主线

本章解决的核心问题是：**向量化方法无法直接处理图结构数据**。解决路径分三步：

1. **图数据特性** — [[4.1-图数据与图学习简介|图]]是节点与边的集合，无法直接套用卷积或全连接。关键代数工具是[[4.1-图数据与图学习简介|拉普拉斯矩阵]] $L = D - A$。图学习任务按层次分为节点级、链接级、图级。
2. **图表征学习** — 在 GNN 出现前，先通过[[4.2-图表征学习|节点嵌入]]（编码器-解码器框架 + [[4.2-图表征学习|随机游走]]）将结构信息映射到低维向量；[[DeepWalk]] 和 [[Node2vec]] 是代表方法。
3. **图神经网络** — 以[[4.3-图神经网络的基本操作|消息传递范式]]统一 GNN 的各种变体（MESSAGE → AGGREGATE → UPDATE）；[[4.4-图神经网络的经典架构|GCN、GraphSAGE、GAT]] 分别以均值聚合、采样聚合、注意力聚合实现这一范式。

> **核心范式**：消息传递是整个 GNN 领域的统一框架，不同架构的本质区别在于 MESSAGE 和 AGGREGATE 函数的具体设计。

## 关键概念

| 概念                               | 一句话                                    |
| :------------------------------- | :------------------------------------- |
| [[4.1-图数据与图学习简介\|拉普拉斯矩阵]]        | $L = D - A$，图谱分析与图卷积的基础工具              |
| [[4.2-图表征学习\|节点嵌入]]              | 将节点映射到 $\mathbb{R}^d$，编码图中的结构相似性       |
| [[4.2-图表征学习\|随机游走]]              | 以游走共现概率定义节点相似性；[[DeepWalk]] / [[Node2vec]] 的核心 |
| [[4.3-图神经网络的基本操作\|消息传递]]         | GNN 的统一计算范式：邻居消息计算 → 聚合 → 节点表示更新       |
| [[4.4-图神经网络的经典架构\|图卷积网络 (GCN)]]  | 谱域卷积的局部近似，以归一化邻居均值更新节点表示               |
| [[4.4-图神经网络的经典架构\|图注意力网络 (GAT)]] | 消息传递中引入注意力权重，自适应聚合不同邻居的贡献              |

## 关键方法

| 方法 | 意义 |
| :-- | :-- |
| [[DeepWalk]] / [[Node2vec]] | 基于[[随机游走]]的[[节点嵌入]]学习；[[Node2vec]] 可控探索深度/广度 |
| GCN 图卷积 | 谱域 → 空域近似，$\tilde{A}$ 归一化邻接矩阵作为聚合算子 |
| 传导式 / 归纳式训练 | 传导式在固定图上半监督学习；归纳式支持泛化到新节点/图 |
