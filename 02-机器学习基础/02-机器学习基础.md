## 主线

机器学习基础的任务是建立从数据到预测的**系统性框架**，核心路径只有三步：

1. **[[假设类]]** 
2. **[[损失函数]]**
3. **[[优化算法]]** — [[梯度下降]]沿[[损失函数]]的梯度反方向更新参数；[[2.3-逻辑回归|过拟合]]时加 L1/L2 正则化项压制参数幅度。

> **统一视角**：三大支柱（[[假设类]]、损失函数、优化）贯穿全课程，后续神经网络、GNN、Transformer 都是在同一框架下不断扩展[[假设类]]的表达能力。

## 关键概念

| 概念                            | 一句话                                    |
| :---------------------------- | :------------------------------------- |
| [[2.1-机器学习简介\|数据集 / 样本 / 特征]] | 模型学习的原材料；特征是输入，标签是"正确答案"               |
| [[2.1-机器学习简介\|损失函数]]          | 衡量预测与真实的差距，为参数优化提供方向                   |
| [[2.2-线性回归\|泛化能力]]            | 从已知数据推及未知数据的能力，模型好坏的根本标准               |
| [[2.2-线性回归\|线性回归]]            | [[假设类]] = 线性函数，损失 = MSE，优化 = [[梯度下降]]      |
| [[2.3-逻辑回归\|逻辑回归]]            | 通过 [[Sigmoid函数]]输出概率，最大化对数似然等价于最小化对数损失 |
| [[交叉熵损失]]                     | 逻辑回归的多分类推广，输出 $K$ 类概率分布，配合交叉熵损失        |
| [[2.3-逻辑回归\|过拟合]]             | 训练好、泛化差；模型复杂度过高时出现                     |

## 关键方法

| 方法 | 意义 |
| :-- | :-- |
| [[梯度下降]] | 沿损失函数梯度反方向迭代更新参数，通用[[优化算法]] |
| 最大似然估计 | 从概率视角构建损失函数，是逻辑回归的理论来源 |
| L2 正则化 | 在损失中加 $\lambda\|w\|_2$ 惩罚，限制参数大小，缓解过拟合 |
| L1 正则化 | 加 $\lambda\|w\|_1$ 惩罚，产生稀疏解，可做特征选择 |
