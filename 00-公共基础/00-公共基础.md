# 00-公共基础

本章作为第 0 章，收录人工智能领域的基础概念与通用函数。这些内容在教材中散落于各章节，此处集中整理，供其他章节统一引用。

---

## 基本概念

| 概念           | 一句话                                                   |
| :------------- | :------------------------------------------------------- |
| [[假设类]]     | 定义了用于拟合数据的候选函数集合，是机器学习三大支柱之一 |
| [[损失函数]]   | 衡量预测与真实标签的差距，为参数优化提供方向             |
| [[优化算法]]   | 在给定假设类和损失函数下，快速找到最优参数的方法         |
| [[正则化方法]] | 在损失中加入参数惩罚项，防止模型过拟合的通用手段         |
| [[激活函数]]   | 神经网络中引入非线性的关键组件，没有它网络等价于线性模型 |
| [[拟合与泛化]] | 训练误差与测试误差之间的权衡，是模型好坏的根本评判标准   |

---

## 算法与函数

| 算法 / 函数     | 一句话                                                           |
| :-------------- | :--------------------------------------------------------------- |
| [[梯度下降]]    | 沿损失函数梯度反方向迭代更新参数，最基础的[[优化算法]]           |
| [[反向传播]]    | 用链式法则高效计算神经网络中所有参数梯度，训练的核心算法         |
| [[Sigmoid函数]] | 将实数压缩到 $(0,1)$，早期神经网络的[[激活函数]]，现多用于输出层 |
| [[Softmax函数]] | 将实数向量转为概率分布，多分类问题的标准输出函数                 |
| [[ReLU]]        | $\max(0,z)$，现代深度网络的主流[[激活函数]]，缓解梯度消失问题    |
| [[L1正则化]]    | 以参数绝对值之和为惩罚项，倾向产生稀疏解，可作特征选择           |
| [[L2正则化]]    | 以参数平方和为惩罚项，将系数向零压缩，防止权重过大               |
| [[残差连接]]    | 将输入直接加到输出上，使深层网络"至少不差于"其浅层子网络         |
